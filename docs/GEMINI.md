# 프로젝트 목표: 중장년/노인 대상 인지건강 & 정서케어 챗봇

## 1. 챗봇의 핵심 기능 정의

### 인지 건강 (Cognitive Health)
- **두뇌 훈련 게임:** 간단한 기억력 게임 (카드 맞추기), 숫자/단어 암기, 초성 퀴즈, 간단한 계산.
- **치매 예방 정보:** 건강한 식습관, 운동법, 인지 활동 관련 정보 제공.
- **일정 및 약 복용 알림:** 사용자의 건강 스케줄을 관리해주는 비서 역할.
- **회상 치료:** "예전에 가장 좋아했던 노래는 무엇이세요?", "어릴 적 살던 동네는 어땠나요?" 등 과거의 긍정적 기억을 자극하는 대화.

### 정서 케어 (Emotional Care)
- **말동무 및 일상 대화:** "오늘 하루 어떠셨어요?", "식사는 하셨나요?"와 같은 일상적이고 다정한 안부 묻기.
- **공감 및 긍정적 피드백:** 사용자의 감정(외로움, 우울함, 기쁨)을 인식하고 공감하며 긍정적인 말을 건네는 기능.
- **관심사 기반 대화:** 사용자가 좋아하는 주제(예: 트로트, 건강, 뉴스, 종교)에 맞춰 대화 유도.
- **가족 연결:** 사용자의 상태나 주요 대화 내용을 요약하여 (사용자 동의하에) 보호자나 가족에게 전달하는 기능.

## 2. 👵 대상 사용자(중장년/노인) 특화 고려사항
- **음성 우선 (Voice-First) 인터페이스:** STT/TTS 기능 핵심.
- **간결하고 직관적인 UI/UX:** 큰 글씨, 높은 명암 대비, 쉬운 용어("말하기", "게임하기") 사용.
- **챗봇 페르소나:** 항상 공손하고(존댓말) 인내심 있으며 사용자를 존중하는 태도. '경청'하는 태도.

## 3. 💻 사용할 기술 스택
- **프론트엔드:** React + Vite
- **UI:** 큰 글씨와 명확한 버튼.
